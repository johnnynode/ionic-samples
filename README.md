# ionic v1 的 tab app 模板演示

## 克隆
- $ `git clone -b ionic-v1 git@github.com:johnnynode/ionic-sample.git  --depth 1`

## 安装
- $ `cd ionic-sample`
- $ `npm i` 或 $ `yarn install`

## 运行
- $ `gulp server`

- 暂未全部完成
